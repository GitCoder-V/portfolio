Технический аудит сайта — moxsmartdisplay.tilda.ws

15 марта 2025

Введение

Поступил сайт для базового технического аудита: https://moxsmartdisplay.tilda.ws/

Задача: Проверить на ошибки и указать проблемные точки.

Технический аудит сайта: moxsmartdisplay.tilda.ws

В ходе аудита были проведены следующие проверки:

  • Проверка редиректа с HTTP на HTTPS
  • Проверка отображения контента при отключенном CSS
  • Тестирование чувствительности к регистру в URL
  • Проверка обработки WWW-версии домена
  • Проверка поведения со слешами
  • Тестирование доступности контента при отключенном JavaScript

1. HTTP → HTTPS редирект

  • Редирект работает корректно на всех страницах
  • Код ответа: 301 Moved Permanently
  • Безопасное соединение настроено правильно

Оценка: ✅ Хорошо

2. Отображение без CSS

  • Контент остается доступным и читаемым
  • Текстовая информация сохраняется
  • Навигация функционирует
  • Поисковые роботы видят весь текстовый контент

Оценка: ✅ Хорошо

3. Чувствительность к регистру в URL

  • При добавлении заглавных букв в URL происходит некорректный редирект
  • Пример: https://moxsmartdisplay.tilda.ws → https://moxsmartdisplay.tilda.ws/indoordisplay?tfc_storepartuid[...]
  • Создание дублей страниц, потеря SEO-веса, непредсказуемое поведение

Оценка: ❌ Критическая ошибка

4. Разделение WWW и non-WWW версий

  • www.moxsmartdisplay.tilda.ws открывается без редиректа на каноническую версию
  • Распределение поведенческих факторов и SEO-веса между двумя версиями сайта

Оценка: ❌ Ошибка

5. Согласованность слешей в URL

  • Неодинаковое поведение со слешами на разных страницах
  • Создание дополнительных дублей контента (версии со слешем и без)

Оценка: ❌ Ошибка

6. Зависимость от JavaScript

  • При отключении JS весь контент страниц пропадает
  • Остается пустая белая страница без текста и навигации
  • Сайт не индексируется поисковыми системами
  • Недоступен для пользователей с отключенным JS

Оценка: ❌ Критическая ошибка

Общие выводы

Несмотря на корректную работу HTTPS и хорошую доступность контента при отключенном CSS, сайт имеет несколько критических проблем:

  1. Чувствительность к регистру — создает дубли страниц
  2. Отсутствие редиректа с WWW — распыление SEO-веса
  3. Несогласованность слешей — дополнительные дубли
  4. Полная зависимость от JavaScript — сайт не индексируется поисковиками

Особенно критична проблема с JavaScript: сайты на Tilda, построенные на визуальном редакторе, часто имеют эту проблему. Для решения необходимо настроить серверный рендеринг или использовать альтернативные платформы, если SEO является приоритетом.
